#### 1. Что выведет программа? Объяснить вывод программы.
   
```go
package main

import (
   "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Выводом будет:
```go
[77 78 79]
```

Создается слайс b с диапазоном значений [1, 4). Элемент с индексом 4 не входит в новый слайс.
